# WallDit
A python program that sets your wallpaper as you choose (GUI version)

# GIF:
![How it works](http://i.imgur.com/oZszzV3.gif)

# Plans:
* Linux support
* Timer to automatically change wallpaper
* Launch on startup

# Setup:
* download the praw module
```
pip download -U praw
```

# Launch:
```
python WallDit.py
```
